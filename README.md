# Recipe to create New_Working_Project environment

1. Create a new project folder (or clone an existing repo) and open it in VS Code
2. Open a terminal and confirm that you are in the project root directory
3. Create a virtual enviornment and activate it
4. Upgrade pip if necessary
5. Install required packages for the project that are needed
6. Save the dependencies to the requirement.txt
7. Add the virtual environment to .gitignore to prevent any further issues
8. In VS Code, select the .venv interpreter
9. Confirm that the environment works
10. Commit changes made and push to GitHub
